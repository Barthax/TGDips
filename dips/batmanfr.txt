Note: standard STV BIOS dual-layer interface.  Service Mode is accessed with the Service button (F2).  Service Mode is navigated with the Test button (9) and selection made with the Service button (F2).  The final Service Mode option is the game cartridge selection which is chosen with the Service Mode navigation.  Entering the cartridge-specific section changes the navigation options to P1 Up and Down.  Selection and alteration is achieved with P1 Start button.  Context-specific navigation instructions are given at the bottom of the screen.
